# Community

Welcome to the Bluetooth SIG Standards Library community!

## Contributing

We welcome contributions of all kinds:

- **Code contributions** - Bug fixes, new characteristics, performance improvements
- **Documentation** - Tutorials, how-to guides, typo fixes
- **Bug reports** - Help us identify and fix issues
- **Feature requests** - Suggest new capabilities

See the **[Contributing Guide](../how-to/contributing.md)** for detailed instructions.

Please read our **[Code of Conduct](code-of-conduct.md)** before participating in the community.
